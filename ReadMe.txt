Olá! Bem vindo(a) ao projeto.

Para executar o sistema com êxito, siga os passos a seguir:

Na pasta "ConsoleBanco", abra o arquivo "App.config". Em seguida, mude os valores da string de conexão, de acordo com a sua própria conexao, mantendo o nome do Database.
Voltando a pasta "AppMenu_Library_GabrielCerruti", abra o script "dbData" e execute no banco de dados. Pronto! Agora é só testar o programa! :)
